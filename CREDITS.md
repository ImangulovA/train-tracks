# Credits

## The puzzle

**Train Tracks** (also known as *Tracks* or *Railroad*) is a logic puzzle
popularised in print and online. This project is an **independent
implementation**: every puzzle here is **generated by our own code**
(`scripts/gen_days.py`) with a solver that guarantees a unique, no-guessing
solution. No third-party puzzle content is bundled or republished.

- The excellent free print puzzles by **Jim Bumgardner / Krazydad**
  (krazydad.com) were used *only as a difficulty reference* while calibrating
  grid sizes per tier. The Krazydad PDFs live in `sources/` locally and are
  **git-ignored** — they are not part of this repository or the deployed site.
- A well-known digital reference for the interaction model is **Simon Tatham's
  "Tracks"** (part of his Portable Puzzle Collection), whose edge-drawing model
  inspired our drag-to-lay-rails input. This project shares no code with it.

## The platform

Built on a small reusable "one puzzle a day" web-game template
(SvelteKit + adapter-static, optional Cloudflare Worker for global stats).
